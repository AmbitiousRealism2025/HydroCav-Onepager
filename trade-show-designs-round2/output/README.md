## How to Use

### Viewing Designs

1. **Interactive Preview:**
   - Open `index.html` in your web browser
   - Browse all 15 designs with live previews
   - Click "View Full" to see complete design
   - Click "Download" to save individual files

2. **Direct Access:**
   - Navigate to any agent folder
   - Open desired HTML file in web browser
   - Each file is fully self-contained and ready to view

### Printing Designs

1. Open any HTML file in a web browser
2. Use browser's Print function (Ctrl+P / Cmd+P)
3. Select "Print to PDF" or physical printer
4. Ensure page size is set to Letter (8.5"×11")
5. Print with margins set to 0.5 inches

**Print Settings Recommendations:**
- Page Size: Letter (8.5"×11")
- Margins: 0.5 inches (already set in CSS)
- Background Graphics: Enabled (to preserve colors and gradients)
- Scale: 100%
